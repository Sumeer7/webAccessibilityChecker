# Example Outputs

This directory contains sample output files generated by the Web Accessibility Checker.

## Files

- **`sample-report.json`** - Detailed JSON report with complete violation information
- **`sample-report.csv`** - CSV format for easy import into spreadsheets
- **`sample-report.html`** - HTML report for viewing in browsers (coming soon)

## Usage

To generate similar reports for your website:

```bash
# JSON output (default)
npx web-accessibility-checker https://example.com

# CSV output
npx web-accessibility-checker https://example.com --format csv

# HTML output
npx web-accessibility-checker https://example.com --format html
```

## Report Contents

Each report includes:
- Summary statistics (violations, passes, incomplete, inapplicable)
- Detailed violation information with:
  - WCAG rule ID
  - Impact level (critical, serious, moderate, minor)
  - Affected elements and CSS selectors
  - Specific failure messages
  - Links to remediation documentation
